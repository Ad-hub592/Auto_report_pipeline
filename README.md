# Auto_report_pipeline
 This makes task easier to manually calculate the Daily Sales Report and sending 
 daily sales reports via email repetitive and error-prone.
 So, this script automatically calculated the Daily Sales Report on the basis of 
 User Input data and create a report that will be sended on the Mail.
# Features
- Sends mail using GMAIL SMTP
- Easily forms Report on the basis of the User Inputs.
- Creates and attaches a report file automatically.
- Uses environment variables for credentials. 
-Error-handling SMTP connection.
# How to Run this 
- Clone the Python script named "main.py"
- Fill the data with your Own Credentials.
- Run:
  - python main.py
- Enter the Products Types and Then enter the further details related to the
   Products.
